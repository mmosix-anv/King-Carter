# Sanity Studio

## Setup

1. Create a Sanity project at https://www.sanity.io/manage
2. Copy `.env.template` to `.env` and add your project ID
3. Run `npm install` from the root directory
4. Run `npm run dev:studio` to start the studio

## Development

- `npm run dev:studio` - Start development server
- `npm run build:studio` - Build for production
